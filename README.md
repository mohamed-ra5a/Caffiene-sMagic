## About the Project

COFFEEO is a responsive café website built with HTML and CSS. It features a welcoming landing page, an introduction section, featured coffee selections, a menu, opening hours, a gallery, and a contact form. The layout is designed to be simple, organized, and easy to navigate on both desktop and mobile devices.








## Features

* Responsive design for desktop and mobile devices
* Smooth scrolling navigation
* Featured coffee offers and menu section
* Opening hours display
* Image gallery
* Contact form
* Custom styling with CSS and hover effects






## Technologies Used

* HTML5
* CSS3
* Google Fonts (Quicksand)
* Font Awesome







## Getting Started

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in your web browser.
